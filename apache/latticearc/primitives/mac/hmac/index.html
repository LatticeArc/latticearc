HMAC (Hash-based Message Authentication Code)

HMAC-SHA256 backed by FIPS 140-3 validated aws-lc-rs.

Standards:

  • RFC 2104: HMAC: Keyed-Hashing for Message Authentication
  • FIPS 198-1: The Keyed-Hash Message Authentication Code (HMAC)
  • NIST SP 800-107: Recommendation for Applications Using Approved Hash Algorithms

The HMAC formula is: H((K ⊕ opad) || H((K ⊕ ipad) || text))

All HMAC operations in the crate share a single backend (aws-lc-rs). Key padding, constant-time tag verification, and FIPS compliance are handled by the underlying library.
