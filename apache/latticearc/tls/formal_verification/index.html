Formal verification support for TLS security properties.

This module provides formal verification capabilities using Kani and SAW. Enable with --features formal-verification, --features kani, or --features saw. Formal verification framework for TLS security properties

This module provides formal verification of critical TLS security paths using Kani model checking, property-based testing, and SAW cryptographic proofs.
