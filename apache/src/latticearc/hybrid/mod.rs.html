1//! Hybrid Cryptography for LatticeArc
2//!
3//! This crate provides hybrid cryptographic schemes that combine post-quantum
4//! and classical algorithms for enhanced security during the quantum transition period.
5//!
6//! # Architecture Overview
7//!
8//! ```text
9//! ┌─────────────────────────────────────────────────────────────────────────┐
10//! │                        HYBRID CRYPTOGRAPHY LAYER                        │
11//! ├─────────────────────────────────────────────────────────────────────────┤
12//! │                                                                         │
13//! │  ┌─────────────────────────────────────────────────────────────────┐   │
14//! │  │                    Hybrid Constructions                         │   │
15//! │  │                                                                 │   │
16//! │  │   ┌───────────────┐  ┌───────────────┐  ┌───────────────────┐  │   │
17//! │  │   │ kem_hybrid    │  │ sig_hybrid    │  │ encrypt_hybrid    │  │   │
18//! │  │   │               │  │               │  │                   │  │   │
19//! │  │   │ ML-KEM-768    │  │ ML-DSA-65     │  │ Hybrid KEM +      │  │   │
20//! │  │   │    +          │  │    +          │  │ AES-256-GCM       │  │   │
21//! │  │   │ X25519        │  │ Ed25519       │  │                   │  │   │
22//! │  │   └───────┬───────┘  └───────┬───────┘  └─────────┬─────────┘  │   │
23//! │  │           │                  │                    │            │   │
24//! │  │           │     XOR          │      AND           │            │   │
25//! │  │           │  Composition     │  Composition       │            │   │
26//! │  │           └──────────────────┴────────────────────┘            │   │
27//! │  │                              │                                 │   │
28//! │  │  ┌───────────────────────────┴───────────────────────────────┐ │   │
29//! │  │  │                    compose module                         │ │   │
30//! │  │  │  - XOR composition proof (breaks EITHER = breaks HYBRID)  │ │   │
31//! │  │  │  - AND composition proof (breaks BOTH = breaks HYBRID)    │ │   │
32//! │  │  └───────────────────────────────────────────────────────────┘ │   │
33//! │  └─────────────────────────────────────────────────────────────────┘   │
34//! │                                    │                                   │
35//! │                                    ▼                                   │
36//! │  ┌─────────────────────────────────────────────────────────────────┐   │
37//! │  │                    arc-primitives (Core Algorithms)             │   │
38//! │  │                                                                 │   │
39//! │  │   ┌──────────────┐  ┌──────────────┐  ┌──────────────────────┐ │   │
40//! │  │   │   ML-KEM     │  │   ML-DSA     │  │     Classical        │ │   │
41//! │  │   │  FIPS 203    │  │  FIPS 204    │  │  X25519, Ed25519     │ │   │
42//! │  │   │  aws-lc-rs   │  │  fips204     │  │  x25519-dalek        │ │   │
43//! │  │   └──────────────┘  └──────────────┘  └──────────────────────┘ │   │
44//! │  └─────────────────────────────────────────────────────────────────┘   │
45//! │                                                                         │
46//! └─────────────────────────────────────────────────────────────────────────┘
47//!
48//! Security Guarantee: Hybrid remains secure if EITHER algorithm is secure
49//! ```
50//!
51//! # Modules
52//!
53//! - [`encrypt_hybrid`](mod@crate::hybrid::encrypt_hybrid) - Hybrid encryption using ML-KEM + AES-256-GCM
54//! - [`kem_hybrid`](crate::hybrid::kem_hybrid) - Hybrid key encapsulation using ML-KEM + X25519
55//! - [`sig_hybrid`](crate::hybrid::sig_hybrid) - Hybrid signatures using ML-DSA + Ed25519
56//! - [`compose`](crate::hybrid::compose) - Formal security proofs for hybrid composition
57//!
58//! # Security Properties
59//!
60//! | Construction      | Composition | Security Guarantee                    |
61//! |-------------------|-------------|---------------------------------------|
62//! | Hybrid KEM        | XOR         | Secure if ML-KEM OR X25519 is secure  |
63//! | Hybrid Signature  | AND         | Secure if ML-DSA AND Ed25519 secure   |
64//! | Hybrid Encryption | XOR (KEM)   | Secure if either KEM component secure |
65
