Module verify 

Source
Expand description

Signature verification with automatic algorithm detection. Verify command — check digital signatures.

Supports two signature formats:

SignedData (recommended): Produced by sign --public-key. Contains scheme metadata, timestamp, and public key — the library’s verify() handles everything. No --algorithm or --key flags needed (both are embedded in the signature).

Legacy JSON: Produced by sign --algorithm. Requires --key (public key) and optionally --algorithm (auto-detected from the signature file).
