RFC Test Vectors

This module validates LatticeArc’s cryptographic implementations against official IETF RFC test vectors for protocol compliance.

§Supported RFCs

  • RFC 7748: X25519 Elliptic Curve Diffie-Hellman
  • RFC 8032: Ed25519 Digital Signatures
  • RFC 8439: ChaCha20-Poly1305 AEAD
  • RFC 5869: HKDF (HMAC-based Key Derivation Function)
