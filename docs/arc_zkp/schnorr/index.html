Schnorr proof system for discrete-log knowledge proofs. Schnorr Zero-Knowledge Proofs

Implements Schnorr’s protocol for proving knowledge of a discrete logarithm without revealing the secret. Uses the Fiat-Shamir heuristic for non-interactive proofs.

§Protocol

Given generator G and public key P = x*G, prove knowledge of x:

  1. Prover picks random k, computes R = k*G
  2. Challenge c = H(G || P || R || context)
  3. Response s = k + c*x
  4. Verifier checks: sG == R + cP

§Security

  • Uses secp256k1 curve (same as Bitcoin/Ethereum)
  • SHA-256 for Fiat-Shamir challenge
  • Constant-time operations where possible
