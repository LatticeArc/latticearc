Struct ZeroizedBytes

pub struct ZeroizedBytes { /* private fields */ }
Expand description

A secure byte container that zeroizes its contents on drop.

§Security Note

Clone is intentionally NOT implemented to prevent creating copies of sensitive data that might not be properly zeroized. If you need to share the data, use as_slice() to get a reference.
