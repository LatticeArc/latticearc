Module hash

Cryptographic hashing (SHA3-256, SHA-256, SHA-512, BLAKE2b-256). Hash command — compute cryptographic fingerprints of files.

A cryptographic hash produces a fixed-size “fingerprint” of a file. Even a single bit change in the input produces a completely different hash output. This makes hashes useful for verifying file integrity.

Supported algorithms:

AlgorithmStandardOutputNotes
SHA3-256FIPS 20232 BDefault. NIST’s newest family.
SHA-256FIPS 180-432 BWidely used (Git, TLS).
SHA-512FIPS 180-464 BLarger output, same family.
BLAKE2b-256RFC 769332 BFast, modern, used by Argon2.

Output formats: hex (default, lowercase) or base64.
