pub fn run_power_up_tests() -> SelfTestResult
Expand description

Run all FIPS 140-3 power-up self-tests

This function runs Known Answer Tests (KATs) for all approved algorithms. According to FIPS 140-3, these tests must pass before any cryptographic operation can be performed.

ยงReturns

  • SelfTestResult::Pass if all tests pass
  • SelfTestResult::Fail(message) if any test fails
