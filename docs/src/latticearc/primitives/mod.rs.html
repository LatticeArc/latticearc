3//! # LatticeArc Primitives
4//!
5//! Core cryptographic primitives for LatticeArc including post-quantum and classical algorithms.
6//!
7//! All algorithms are always available. Algorithm selection is handled at runtime via
8//! the [`unified_api`](crate::unified_api) auto/context-based selection based on security requirements and hardware
9//! capabilities.
10//!
11//! ## Feature Flags
12//!
13//! - **`fips-self-test`** - Enable FIPS 140-3 power-up self-tests (KAT verification)
14//!
15//! ## Algorithms
16//!
17//! ### Post-Quantum (NIST FIPS 203-206)
18//!
19//! - **kem::ml_kem**: ML-KEM (FIPS 203) Key Encapsulation via `aws-lc-rs`
20//! - **sig::ml_dsa**: ML-DSA (FIPS 204) Digital Signatures via `fips204` crate
21//! - **sig::slh_dsa**: SLH-DSA (FIPS 205) Hash-based Signatures via `fips205` crate
22//! - **sig::fndsa**: FN-DSA (draft FIPS 206) Lattice Signatures via `fn-dsa` crate
23//!
24//! ### Symmetric Encryption (AEAD)
25//!
26//! - **aead::aes_gcm**: AES-GCM-128/256 (NIST SP 800-38D) via `aws-lc-rs`
27//! - **aead::chacha20poly1305**: ChaCha20-Poly1305 (RFC 8439)
28//!
29//! ### Hashing
30//!
31//! - **hash**: SHA-2 (SHA-256, SHA-384, SHA-512) per FIPS 180-4
32//! - **hash**: SHA-3 (SHA3-256, SHA3-384, SHA3-512) per FIPS 202
33//!
34//! ### Key Derivation
35//!
36//! - **kdf::hkdf**: HKDF (RFC 5869, NIST SP 800-56C)
37//! - **kdf::pbkdf2**: PBKDF2 (NIST SP 800-132)
38//!
39//! ### Classical Cryptography
40//!
41//! - **kem::ecdh**: X25519 key exchange (RFC 7748)
42//! - **ec::ed25519**: Ed25519 signatures (RFC 8032)
43//! - **ec::secp256k1**: secp256k1 signatures (Bitcoin/Ethereum compatible)
44//!
45//! ### Supporting Modules
46//!
47//! - **keys**: Hybrid keypair management (ML-KEM + X25519)
48//! - **rand**: Cryptographically secure random number generation
49//! - **mac**: HMAC (FIPS 198-1), CMAC (NIST SP 800-38B)
50//! - **security**: Secure memory containers with zeroization
51//!
52//! ## FIPS 140-3 Compliance Notes
53//!
54//! | Algorithm | Implementation | FIPS Validated |
55//! |-----------|----------------|----------------|
56//! | ML-KEM | `aws-lc-rs` | Yes (Cert #4631, #4759, #4816) |
57//! | ML-DSA | `fips204` crate | No (aws-lc-rs API not yet stable) |
58//! | SLH-DSA | `fips205` crate | No |
59//! | FN-DSA | `fn-dsa` crate | No |
60//! | AES-GCM | `aws-lc-rs` | Yes |
61//! | SHA-2/3 | `sha2`/`sha3` crates | No (RustCrypto, widely reviewed) |
62//!
63//! See the `docs/` directory for full compliance details.
64
