6//! # Key Encapsulation Mechanisms (KEM)
7//!
8//! This module provides a unified interface for both post-quantum and classical
9//! key encapsulation mechanisms. It supports the following algorithms:
10//!
11//! ## Post-Quantum Algorithms
12//!
13//! - **ML-KEM (FIPS 203)**: Module-Lattice-Based Key Encapsulation Mechanism
14//!   - ML-KEM-512: NIST Security Category 1 (AES-128 equivalent)
15//!   - ML-KEM-768: NIST Security Category 3 (AES-192 equivalent)
16//!   - ML-KEM-1024: NIST Security Category 5 (AES-256 equivalent)
17//!
18//! ## Classical Algorithms
19//!
20//! - **ECDH (X25519)**: Elliptic Curve Diffie-Hellman key exchange (RFC 7748)
21//!
22//! ## Security Properties
23//!
24//! All KEM implementations in this module provide:
25//! - **IND-CCA2**: Indistinguishability under adaptive Chosen-Ciphertext Attack
26//! - **Constant-time**: All secret-handling operations execute in constant time
27//! - **Zeroization**: Secret keys are securely wiped when dropped
28//!
29//! ## Example Usage
30//!
31//! ### ML-KEM Key Encapsulation
32//!
33//! ```no_run
34//! use arc_primitives::kem::ml_kem::{MlKem, MlKemSecurityLevel};
35//! use rand::rngs::OsRng;
36//!
37//! // Generate keypair
38//! let mut rng = OsRng;
39//! let (pk, sk) = MlKem::generate_keypair(&mut rng, MlKemSecurityLevel::MlKem768)?;
40//!
41//! // Encapsulate shared secret
42//! let (shared_secret, ciphertext) = MlKem::encapsulate(&mut rng, &pk)?;
43//!
44//! // Decapsulate shared secret
45//! let recovered_secret = MlKem::decapsulate(&sk, &ciphertext)?;
46//! assert_eq!(shared_secret, recovered_secret);
47//! # Ok::<(), Box<dyn std::error::Error>>(())
48//! ```
49//!
50//! ## Mathematical Correctness
51//!
52//! The ML-KEM implementation is based on the Module-LWE (Learning with Errors)
53//! hardness problem, which provides provable quantum resistance. The security
54//! reduction ensures that breaking ML-KEM requires solving the Module-LWE problem,
55//! which is believed to be hard even for quantum computers.
56//!
57//! ## Module Structure
58//!
59//! - [`ml_kem`]: ML-KEM (FIPS 203) post-quantum KEM
60//! - [`ecdh`]: Classical elliptic curve Diffie-Hellman (X25519)
