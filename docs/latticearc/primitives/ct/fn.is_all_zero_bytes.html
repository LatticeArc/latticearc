pub fn is_all_zero_bytes(bytes: &[u8]) -> bool
Expand description

Returns true iff every byte of bytes is zero, in constant time.

Empty slices return false so that a caller passing a zero-length secret cannot trigger a false-positive weak-input rejection.

§Constant-time behaviour

Runtime is independent of the position of the first non-zero byte. Internally compares bytes against an all-zero stack buffer in 32-byte chunks via subtle::ConstantTimeEq::ct_eq; longer inputs (e.g. PBKDF2 salts greater than 32 bytes) are handled chunk-by-chunk without falling back to a heap allocation, and without an early exit on chunk boundaries (each chunk’s Choice is &-folded).
